자료구조 과제 2 · OOP Polymorphism

스마트 기기 상속 구조와 다형성 시각화

과제 1의 DeviceADT → Device → AirConditioner / RobotVacuum 구조를 웹 화면에 시각화했습니다. 각 노드를 클릭하면 실제 JavaScript 클래스 객체의 operate() 메서드가 호출되고, 하위 클래스마다 오버라이딩된 결과가 다르게 출력됩니다.

ADTDeviceADT 추상 인터페이스
BaseDevice 공통 속성/메서드
Instances에어컨, 로봇청소기

사용 방법

왼쪽 상속 구조에서 카드를 클릭하세요. 같은 이름의 operate() 메서드를 호출해도 객체의 실제 클래스에 따라 다른 동작이 실행됩니다.

상속 구조 시각화

카드 클릭 → 메서드 호출

핵심 구현 코드

class Device {
  operate() {
    return `${this.name} 기기가 작동합니다.`;
  }
}

class AirConditioner extends Device {
  operate() {
    return `${this.name} 에어컨이 시원한 바람을 냅니다.`;
  }
}
